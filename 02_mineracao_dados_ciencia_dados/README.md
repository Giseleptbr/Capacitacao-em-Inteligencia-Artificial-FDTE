# 02 - Fundamentos de Mineração de Dados e Ciência de Dados (FMC)

Esta pasta contém os **notebooks desenvolvidos durante a disciplina Fundamentos de Mineração de Dados e Ciência de Dados**, parte do programa **Capacitação em Inteligência Artificial da FDTE (2025)**.

A disciplina apresenta os conceitos fundamentais de **mineração de dados, modelagem preditiva e avaliação de modelos**, além de introduzir técnicas utilizadas em projetos de ciência de dados.

---

# Conteúdos abordados na disciplina

Durante o curso foram explorados conceitos teóricos e práticos relacionados à mineração de dados e análise de dados em diferentes contextos.

Entre os principais tópicos abordados estão:

- fundamentos de mineração de dados
- coleta e preparação de dados
- processamento de dados
- análise exploratória de dados
- modelagem de dados com algoritmos de classificação
- árvores de decisão (**CART**)
- **Random Forest (Florestas Aleatórias)**
- avaliação de desempenho de modelos
- mineração de dados em larga escala (**Big Data**)
- algoritmos para **data streams**
- regressão para fluxos de dados
- privacidade e proteção de dados

As atividades foram desenvolvidas utilizando **Python e Jupyter Notebooks**, com exercícios aplicados a diferentes conjuntos de dados. :contentReference[oaicite:0]{index=0}
