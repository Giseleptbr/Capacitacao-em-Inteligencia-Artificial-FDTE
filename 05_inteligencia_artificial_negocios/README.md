# Inteligência Artificial Aplicada aos Negócios (IAN)

Disciplina do curso **Capacitação em Inteligência Artificial – FDTE**.

## Conteúdos abordados

- Introdução à Inteligência Artificial aplicada ao negócio
- Análise de dados com a base Olist
- Construção de personas
- Design centrado no usuário
- Brainstorming e prototipagem
- Desenvolvimento e apresentação de soluções baseadas em dados


## Tópicos estudados

### Pesquisa e entendimento do problema
- coleta de dados
- análise qualitativa e quantitativa
- identificação de padrões de comportamento

### Design centrado no usuário
- construção de **personas**
- mapa de empatia
- jornada do usuário

### Desenvolvimento de soluções
- brainstorming
- benchmarking
- prototipagem de soluções
- priorização de ideias

## Base de dados utilizada

- Dataset **Olist E-commerce**
- Análise exploratória e desenvolvimento de soluções baseadas em dados:
  
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
