# Processamento de Imagem e Visão Computacional (PIV)

Disciplina do programa **Capacitação em Inteligência Artificial – FDTE**.

Este módulo aborda os principais fundamentos de **processamento digital de imagens e visão computacional**, explorando técnicas clássicas de análise de imagens, filtragem, segmentação e detecção de objetos.

Os notebooks deste diretório apresentam implementações práticas utilizando Python e bibliotecas de visão computacional.

---

# Conteúdos da disciplina

Os principais tópicos abordados foram:

- Introdução à Visão Computacional
- Processamento digital de imagens
- Equalização de histograma
- Filtros lineares
- Filtros não lineares
- Morfologia matemática
- Segmentação de imagens
- Detecção de objetos
- Reconhecimento de padrões

As atividades práticas foram desenvolvidas ao longo das aulas utilizando conjuntos de imagens fornecidos durante o curso. :contentReference[oaicite:1]{index=1}

- espectro de frequência
- filtragem no domínio da frequência
- interpretação de padrões espaciais
